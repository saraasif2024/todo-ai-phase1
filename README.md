# Todo CLI App - Phase 1

This is a command-line based Todo application built as part of an AI-driven development project.

## Features
- Add, View, Update, and Delete tasks.
- Mark tasks as completed.
- Permanent storage using a JSON file.
- Built with Python 3.13 in a WSL (Ubuntu) environment.

## How to Run
1. Ensure you have Python 3.13 installed.
2. Run `python3 todo.py add "Your Task"` to add a task.
3. Use `python3 todo.py view` to see your list.
