# Todo App Specification - Phase 1

Create a command-line Todo application using Python 3.13.

## Requirements:
- Use a JSON file to store tasks.
- Each task should have: ID, Description, Status (Pending/Completed).

## Features to Implement:
1. **Add Task:** User can add a new task.
2. **View Tasks:** Show a list of all tasks.
3. **Delete Task:** Remove a task using its ID.
4. **Update Task:** Edit the description of an existing task.
5. **Complete Task:** Mark a task as done.
